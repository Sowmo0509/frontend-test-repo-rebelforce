## Upload Issues

### Some uploads feel stuck for a long time

**Sprint Points:** 5

- Most files upload quickly.
- Occasionally, an upload spins for a very long time.
- We are assuming these uploads are failing when this happens.

---

### Newly uploaded documents don’t always show up

**Sprint Points:** 8

- After uploading a document, it doesn’t always appear in the documents list immediately.
- Users will almost certainly assume the upload failed.
- Annual Reports and Compliance Reports uploads seem particularly buggy.

---

## Document Actions Issues

### Approving or deleting documents shows errors

**Sprint Points:** 3

- When attempting to approve or delete documents, a red “something went wrong” error message often appears.
- The root cause is currently unknown.

---

### Same error appears across registration and other actions

**Sprint Points:** 3

- Multiple actions (registration, approving, deleting, etc.) show the same error message.
- This happens even though the contexts are different, making debugging and UX confusing.

---

## UI / Responsiveness Issues

### Documents and Users pages behave awkwardly on different screen sizes

**Sprint Points:** 3

- Issues appear when resizing the browser.
- Layout problems are noticeable on smaller screen sizes.
